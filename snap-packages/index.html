Snapcrafters dashboard

This page shows the status of all our snap packages. It’s updated every 15 minutes using releasegen.

The goal of this page is to give snapcrafters and community members a clear overview of what work still needs to be done. Take a look at our contributing guide for more information on how to contribute.

An easy way to start contributing is to test snaps and review pull requests.

  • Open calls for testing
  • Open pull requests

Status

 

  • For a release to show here, it must be part of one of the `snapcrafters` GitHub organisation. GitHub repositories must not be Archived. The latest three releases are shown for each repository.

    For GitHub badges to be shown for a repository, add them in the README; similarly, to display the extra Snap Store release information in the expanding table rows, add the Snap Store badge ( ![](https://snapcraft.io/*snap-name*/badge.svg) ) to the snap's README.
